Introduction

AstaIO is one source compatible for Delphi 5, Delphi 6, Delphi 7 and Kylix 2 and 3. AstaNative Sockets are written to the WinSocket and Linux Socket API. By default AstaIO servers run on port 9050.

This build includes SOAP support, Servers, Tutorials, a new 600+ Page PDF Manual and an updated AstaIO help file.

Updated AstaIO Information can always be found from the AstaIO Page on the ASTA Web Site.

Installation

AstaIO requires 2 packages to be installed an AstaSOAP package and then the AstaIO package.

For Delphi 6 first install Asta_SoapD6.bpl (which includes the ASTA XML Parser, Soap Units and AstaIO Base Sockets) and then browse and install AstaIOD6.bpl

Important Changes

This build includes all the architectural changes for nestedDataSet support and we are now testing selects and updates with nestedDataSets and working on tutorials.

An AstaIO ODBC driver available as part of the AstaIO Safari Kit.

New Features

This build includes:

  • ExecSQLTransaction
  • Multiple Parameterized Queries from TAstaIOClientQueries
  • AstaIOClientDataSet and AstaIODataSetProvider
  • NestedDataSet Support
  • Windows Server format that allows for servers to be run as NT services or Normal EXE's. this build contains ADO, IBObjects and IbExpress examples using this format.
